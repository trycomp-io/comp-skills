# ceo-people-update-drafter

Skill gratuito do Claude, o Chief of Staff do CHRO. Gera o update CHRO → CEO (mensal/trimestral) em formato 1-pager estruturado.

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-ceo-people-update-drafter)).

## Instalação

```bash
/plugin marketplace add trycomp-io/comp-skills
/plugin install comp-skills@comp
```

## Uso

```
"Drafta meu update mensal pro CEO"
"People update do Q2, me ajuda a estruturar"
"Update do CHRO pro fundador"
```

## Estrutura

- Resumo executivo
- Métricas-chave (com trend + contexto)
- Movimentações (hires, promoções, exits)
- Wins
- Riscos (com mitigação + owner)
- Asks pro CEO

## O que é compartilhado

Email opcional + telemetria opt-in. **Nunca** envia conteúdo do update.
