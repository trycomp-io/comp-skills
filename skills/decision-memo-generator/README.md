# decision-memo-generator

Skill gratuito do Claude, o Chief of Staff do CHRO. Gera memo estruturado de decisão pra qualquer people topic crítico.

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-decision-memo-generator)).

## Instalação

```bash
/plugin marketplace add trycomp-io/comp-skills
/plugin install comp-skills@comp
```

## Uso

```
"Estrutura um decision memo sobre aumentar headcount no H2"
"Memo pro CEO sobre reorg do time de Eng"
"Frame essa decisão de mudar a política de remote"
```

## Estrutura

- Problema
- Contexto
- 2-3 opções (com prós/cons)
- Recomendação com rationale
- Riscos
- Ask

## O que é compartilhado

Email opcional + telemetria opt-in. **Nunca** envia conteúdo do memo.
