# promotion-equity-analyzer

Skill gratuito do Claude. Analisa equidade de promoções por gênero: gap F vs M, áreas com disparidade, transições de nível.

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-promotion-equity-analyzer)).

## Instalação

```bash
/plugin marketplace add trycomp-io/comp-skills
/plugin install comp-skills@comp
```

## Uso

```
"Analisa as promoções do último ano por gênero"
"Tem gap de promoção entre F e M?"
"Equidade de promoção por área"
```

## CSVs necessários

- Promoções (obrigatório): `gender` no mínimo
- Eligible population (recomendado): pra calcular taxas reais

## O que é compartilhado

Email opcional + telemetria opt-in. **Nunca** envia dados de colaborador.
