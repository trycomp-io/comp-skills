# board-people-slide-builder

Skill gratuito do Claude que gera o slide de People & Culture pra board meeting em HTML 16:9 (printable pra PDF).

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-board-people-slide-builder)).

## Instalação

```bash
/plugin marketplace add trycomp-io/comp-skills
/plugin install comp-skills@comp
```

## Uso

```
"Prepara o slide de People pro board do Q2"
"Board deck: preciso de uma página de update de people"
"Gera slide com headcount, attrition, eNPS e diversity"
```

## Estrutura

- 4 KPIs principais (com trend e contexto)
- 3 highlights narrativos
- Riscos e asks separados

## Output

HTML 16:9 (1920x1080). Abra no browser → imprima como PDF → inclua no board deck. Ou exporte como imagem.

## O que é compartilhado

Email opcional + telemetria opt-in. **Nunca** envia conteúdo do slide.
