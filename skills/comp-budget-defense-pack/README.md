# comp-budget-defense-pack

Skill gratuito do Claude, o Chief of Staff do CHRO. Gera pacote defensável de comp budget pra apresentar ao CFO/CEO.

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-comp-budget-defense-pack)).

## Instalação

```bash
/plugin marketplace add trycomp-io/comp-skills
/plugin install comp-skills@comp
```

## Uso

```
"Prepara um pacote pra defender meu comp budget do H2"
"Justificar reajuste de 5% no time de Eng pro CFO"
"Defense pack: 8 novas vagas + dissídio + 3 ajustes pontuais"
```

## Estrutura

- Sumário: pedido total + % impacto na folha
- Linhas justificadas (categoria, item, HC, custos com encargos, rationale, benchmark)
- Cenários alternativos (100%, 50%, 25%)
- Riscos se negado
- Ask cristalino

## O que é compartilhado

Email opcional + telemetria opt-in. **Nunca** envia conteúdo do pack.
