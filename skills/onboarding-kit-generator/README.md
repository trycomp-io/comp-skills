# onboarding-kit-generator

Skill gratuito do Claude para líderes de RH & People. Gera um kit completo de onboarding personalizado (plano 30/60/90, checklist IT, 1:1s estratégicos, email de boas-vindas, script do buddy, template de 1:1 do manager) em HTML printable + Markdown editável.

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-onboarding-kit-generator)).

## O que faz

Você descreve o cargo + contexto. Claude gera o conteúdo personalizado (não genérico) e renderiza dois arquivos:
- **HTML**: pronto pra abrir no navegador, printar ou compartilhar
- **Markdown**: editável, fácil de ajustar no Notion/Docs/Slack

## Instalação

```bash
/plugin marketplace add trycomp-io/comp-skills
/plugin install comp-skills@comp
```

## Uso

```
"Prepara o onboarding kit pra Eng Manager L5 começando dia 15/06, manager é Maria"
"Gera um plano 30/60/90 pra Head of Marketing que entra mês que vem"
"Checklist de onboarding pra senior designer entrando no time"
```

Quanto mais contexto você der (manager, buddy, particularidades do time), mais customizado o kit fica.

## O que vai no kit

| Seção | Conteúdo |
|---|---|
| Plano 30/60/90 | Aprender (30) → Contribuir (60) → Liderar (90), específico pro cargo |
| Checklist IT | Equipamentos, acessos, integrações |
| 1:1s estratégicos | Quem encontrar nas primeiras semanas e pra quê |
| Email de boas-vindas | Pronto pra ajustar e enviar |
| Script do Buddy | Role + cadência sugerida |
| Template de 1:1 do Manager | Estrutura + perguntas-gatilho pras primeiras 4 semanas |

## O que é compartilhado com a Comp

Email opcional + telemetria opt-in. **Nunca** envia conteúdo do kit ou dados do colaborador.
