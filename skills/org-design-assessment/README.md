# org-design-assessment

Skill gratuito do Claude para líderes de RH & People. Assessment HTML de maturidade em design organizacional, baseado no [artigo Comp na Cajuína](https://cajuina.org/principais/coluna-comp/design-organizacional/).

Mantido pela **Comp** ([comp.vc](https://comp.vc?utm_source=github&utm_medium=readme&utm_campaign=eam&utm_content=skill-org-design-assessment)).

## O que faz

12 perguntas em 4 pilares (~5 min). Output: Score 0-100 por pilar + geral + classificação + próximo passo personalizado.

4 pilares (do framework Comp):
1. **Maturidade de Governança**: modelos contratuais como alavanca estratégica
2. **Span of Control**: eficiência de liderança
3. **Pirâmide de Níveis**: composição vs estratégia
4. **Disparidade Salarial Interna**: equidade (80%-120% mediana)


## Classificação

| Score | Estágio |
|---|---|
| 0-40 | Reativo (decisões na intuição) |
| 41-60 | Inicial |
| 61-75 | Maduro (decisões informadas por dados) |
| 76-90 | Avançado |
| 91-100 | Estratégico (org design como alavanca de negócio) |
